# 产品概述

douyin-crawler 是一个多平台视频/图集解析下载服务。

## 核心功能

- 解析多平台分享链接，提取视频或图集信息（标题、封面、作者等）
- 支持抖音、小红书、快手、B站等主流平台
- 支持视频流式下载（无水印）
- 支持图集浏览和下载
- 提供代理下载接口，转发远程文件流到客户端

## 用户界面

Web 前端页面，用户粘贴分享内容后一键解析和下载。前端使用 Vue 3 CDN + EJS 模板渲染。

## API 接口

- `POST /api/parse` — 解析分享链接，返回视频/图集元数据
- `GET /api/download-stream` — 流式下载视频
- `GET /api/proxyFile` — 代理下载远程文件

## 运行端口

默认监听 `7777`，服务端超时设置为 10 分钟。
